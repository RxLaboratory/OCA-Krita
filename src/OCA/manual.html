OCA - Open Cel Animation exporter for Krita

OCA is an open format to ease the exchange of traditionnal/frame-by-frame/cel animation between different applications.

To Export to the OCA format, go to Tools ‣ Scripts ‣ OCA Export.

To learn more about OCA, visit rainboxlab.org.

To learn how to use and what the OCA exporter for Krita can do, visit oca-krita-docs.rainboxlab.org.
